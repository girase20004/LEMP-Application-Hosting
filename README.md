# LEMP-Application-Hosting
Hosted a LEMP stack application on AWS using EC2 and RDS with Linux, Nginx, MySQL, and PHP for fast and efficient web hosting.


## 🎯 Purpose
Host a modern web application using the LEMP stack.

## 🧰 AWS Services Used
- Amazon EC2
- Amazon RDS

## 📌 Project Overview
This project demonstrates hosting a LEMP application using Linux, Nginx, MySQL, and PHP on AWS infrastructure.

## 🚀 Features
- Nginx web hosting
- MySQL database integration
- PHP application support
- Faster web performance
